# Abu Rifath Jahan Portfolio

Modern, responsive digital journalist portfolio built with React + Vite.

## Stack

- React + Vite
- TailwindCSS
- Framer Motion
- React Router

## Theme Direction

- Clean newsroom aesthetic
- White background with black typography
- Red accent highlights
- Mobile-first responsive layout

## Included Pages

- Home: Hero section with journalist profile, CTA buttons, and social links
- Work: Sample story cards for portfolio presentation
- 404: Fallback route page

## Run Locally

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
npm run preview
```
